2. CONTROLLERS DEVONO ESSERE TUTTE CLASSI
3. Ogni calsse dentro un file, il nome della classe deve essere uguale al nome del file e le classi hanno tutte le maiuscole

5. I file di configurazione dovrebbero tornare array
6. La connessione deve essere su una classe globale: guardare pattern Singletone
7. Guarda Composer
